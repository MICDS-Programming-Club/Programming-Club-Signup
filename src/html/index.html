Who can join Programming Club?

Anybody can join Programming Club! No experience is required! Even if you're just interested, don't hesitate to register!

When does Programming Club meet?

Programming Club will usually meet for the full activities period every Day 6, but we can also communicate via an application called Slack. Slack is basically like Skype or Discord, which allows us to work on projects together outside of class.
